# ysoserial-for-woodpecker

## 0x01 简介
`ysoserial-for-woodpecker`是基于 [ysoserial](https://github.com/frohoff/ysoserial) ,为woodpecker框架定制开发的gadget集成库。

## 0x02 功能
- [ ] sleep
- [ ] dnslog
- [ ] httplog
- [ ] uploadfile
- [ ] uploadfile_append
- [ ] general_cmd 常规执行命令
- [ ] auto_cmd  自动判断操作系统执行命令
- [ ] win_cmd windows下执行命令
- [ ] linux_cmd linux下执行命令
- [ ] classfile
- [ ] classbase64
- [ ] bcelfile
- [ ] bcel
- [ ] jsfile
- [ ] jsbase64
- [ ] loadjar
- [ ] jndi
