# ysoserial-for-woodpecker

## 0x01 简介
`ysoserial-for-woodpecker`是基于 [ysoserial](https://github.com/frohoff/ysoserial) ,为woodpecker框架定制开发的gadget集成库。

## 0x02 功能
- [ ] sleep
- [ ] dnslog
- [ ] httplog
- [ ] upload_file
- [ ] raw_cmd 原生的命令执行
- [ ] win_cmd 在windows下执行命令
- [ ] linux_cmd 在linux下执行命令
- [ ] auto_cmd  自动判断操作系统执行命令
- [ ] class_file
- [ ] class_base64
- [ ] bcel
- [ ] bcel_file
- [ ] script_file
- [ ] script_base64
- [ ] loadjar
- [ ] jndi
